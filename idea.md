Build a high-end, visually stunning personal Prompt Database web application. The application should serve as a private vault for AI prompts with the following core features and design language:

Core Functionality:

Prompt Management: Create, read, update, and delete (CRUD) functionality for prompts.

One-Click Copy: Each prompt entry must have a prominent, easy-to-use 'Copy to Clipboard' button.

Categorization: A robust system to organize prompts into custom categories (e.g., Coding, Creative Writing, Data Analysis).

Metadata Tracking: Every prompt entry must include specific fields for:

Model: (The specific AI model used, e.g., GPT-4o, Claude 3.5 Sonnet).

Environment: (Where it was executed, e.g., VibeCode, ChatGPT, API).

Rating System: An interactive 1-5 star efficiency/effectiveness scale for each prompt.

Search & Filter: Quick search bar and filtering by category or rating.

Visual Design & UI/UX:

Modern Aesthetic: Implement an ultra-modern, 'dark-mode first' aesthetic using Glassmorphism (frosted glass effects, subtle blurs).

Visual Impact: Use high-fidelity UI components, smooth CSS transitions, and micro-interactions.

Layout: A sophisticated dashboard-style layout with a sidebar for categories and a main grid or list view for the prompts.

Typography: Clean, professional sans-serif typography with clear hierarchy.

Details: Ensure high attention to detail in shadows, border gradients, and empty states.

Technical Constraints:

Prioritize a responsive layout that looks exceptional on both desktop and mobile.

Ensure the code is modular and easy to extend.
